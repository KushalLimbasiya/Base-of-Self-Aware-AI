# Logs Directory

This directory contains log files generated by Jarvis.

## Log Files

- `jarvis.log` - Main application log with all events, errors, and debug information

## Log Levels

- **DEBUG**: Detailed diagnostic information
- **INFO**: General informational messages
- **WARNING**: Warning messages (e.g., low confidence predictions)
- **ERROR**: Error messages (e.g., API failures)
- **CRITICAL**: Critical errors that cause the application to stop

## Viewing Logs

To view the latest logs:
```bash
# Windows PowerShell
Get-Content logs\jarvis.log -Tail 50

# Linux/Mac
tail -f logs/jarvis.log
```

## Log Rotation

Logs are appended to the same file. To prevent large files, consider implementing log rotation or manually clearing old logs.
